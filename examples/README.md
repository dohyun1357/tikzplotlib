# Examples

[`showcase.ipynb`](showcase.ipynb) walks through a range of matplotlib plots —
line plots, scatter with a colormap, bar charts, histograms, subplots,
logarithmic axes, `clean_figure`, and the LaTeX preamble — and shows the
TikZ/PGFPlots code tikzplotlib generates for each. It is committed with its
outputs so it renders directly on GitHub.

To run it yourself:

```sh
uv run --with jupyterlab jupyter lab examples/showcase.ipynb
```
